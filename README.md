# AutoFlow - No-Code Browser Automation

## MVP Features
1. Auto-discover all pages from main URL
2. Detect login forms → pause for credentials
3. Build workflow automatically
4. User customizes workflow
5. Replay workflow anytime

## Tech Stack
- Python + Playwright
- Simple JSON workflow storage
- Web UI (Flask/FastAPI)

## Usage
```bash
python autoflow.py https://example.com
```
